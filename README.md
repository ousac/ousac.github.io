# OUSAC Website

The official website for the 2nd Annual Ontario Universities Sports Analytics Conference (OUSAC), hosted at the University of Toronto.

## Overview

This project is a static website built with Next.js, designed to provide information about the conference, speakers, schedule, and sponsors. It serves as the central hub for student registration and event details.

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS V4
- **Icons:** Lucide React
- **Fonts:** Inter & League Spartan (via `next/font`)
- **Deployment:** GitHub Pages

## Key Features

- **Responsive Design:** Optimized for mobile and desktop viewing.
- **Performance:** Static site generation with optimized images (`next/image`).
- **Modern UI:** Custom design system with OUSAC branding colors.
- **Accessibility:** Semantic HTML and accessible navigation.

## Project Structure

The project follows a standard Next.js App Router structure within a `src/` directory:

- `src/app`: Page routes and layouts.
- `src/components`: Reusable UI components (Navbar, Footer).
- `src/lib`: Utility functions and constants.
- `src/types`: TypeScript definitions.
- `public`: Static assets (images, logos).
